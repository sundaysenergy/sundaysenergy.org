As ubercart develops and features are added, we will install the new version and enable those features for you. This package includes email/phone support to help you learn to use the new features. We will backup your entire site before doing the upgrade.

Does NOT include compatibility guarantees for contributed modules. Other modules may break or need to be re-written upon upgrading. Ask for details and pricing. Frequency and hours of support are up to the discretion of SE. This package includes reporting any problems/issues (bugs) to the Ubercart community.

This product needs to be purchased for a minimum of one year, and implemented on a site with the latest Ubercart vesion (branch 1 or 2) installed.  On an existing site, Ubercart will need to be updated to the current version first before this product can be purchased.
