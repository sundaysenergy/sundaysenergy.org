Custom Website Feature

Often we get the request to add a feature onto a website. Because of the nature of open-source someone else has probably thought to ask for a similar feature and had it submitted it back to the community so anyone can download and use it for free. This page attempts to explain how our pricing structure works. Keep in mind that to get to the next level all previous levels must have been purchased.

8.1.1Quality Assurance (QA) Testing, Verification, De-bugging

If you discover any problems and require a fix ASAP you need this product. With this service contract, Sundays Energy will track down the problem, report the bugs as needed to the community AND work to solve the issue ourselves. The patch created that fixes the issue will be submitted back into the Drupal community. If you are using this service as a wildcard against all the modules installed on your site, we will fix one bug per month. If you are using this service against one specific module, we will fix any and all bugs that arise from that specific module.
